港区苹果账号 Apple ID 注册教程

2024-07-27

1、注册 Apple ID 苹果账号

1.1 首先，我们打开浏览器，访问 https://appleid.apple.com，然后点击右上角的“创建您的 Apple ID”按钮，进入注册苹果账号页面。如下图所示：

1.2 在注册页面，按要求填写姓名、出生日期、邮箱、密码等信息，国家或地区选择香港，填写手机号码，如无香港手机号码，可以填写“+86”的中国大陆手机号码。然后，选择手机号码验证方式，如短信和语音来电。如下图所示：

1.3 根据注册页面提示，在右侧输入框填入左侧的图片验证码，然后点击下方“继续”按钮，进入下一步。如下图所示：

1.4 登录我们注册用的电子邮箱，查看接收到验证码的电子邮件，将获得的验证码输入文本框，最后点击右下角的“继续”按钮。如下图所示：

1.5 将注册手机接收到的短信验证码输入到文本框，点击“继续”按钮。如下图所示：

1.6 验证成功后，我们就来到苹果 Apple ID 管理界面。如下图所示：


至此，Apple ID 已经基本注册完成，但尚未激活iTunes商店账号，无法正常使用 App Store 商店服务。

2、登录苹果 App Store 商店，并激活 iTunes 服务

2.1 我们在苹果手机iPhone或平板iPad上，打开 App Store，然后使用新注册的 Apple ID 登陆苹果商店。如下图所示：

2.2 在 App Store 输入 Apple ID 和密码，然后将接收到的手机验证码填入输入框。如下图所示：

2.3 在弹出来的窗口中，点击“检查”按钮。如下图所示：

2.4 在“完成创建 Apple ID”页面，启用“同意条款与条件”选项按钮。如下图所示：

2.5 进入“完成设定 Apple ID”页面，付款方式和付款人姓名均保持默认（如果没有付款需求，千万别选择付款方式，否则会无法消除此选项，需要退出再登录才能消除）。然后在“账单地址”项，填入香港账单地址和电话（此处使用的街道地址和电话可在网站 https://www.meiguodizhi.com/hk-address 生成，然后使用）。以上信息填写完毕后，请点击右上角的“下一页”按钮，既可完成注册流程。

2.6 当 Apple ID 建立完成，会看到可使用所有 Apple 服务的提示页面。如下图所示：
