Hiddify客户端使用教程

2024-07-27

1、Hiddify客户端简介

Hiddify，以前名字是 Hiddify NEXT，是一款开源程序，基于 Sing-box 通用代理工具的跨平台代理客户端，而且原生支持多国语言界面，还没有广告。Hiddify跟Clash系列客户端非常相像，不知道是否有渊源，但Hiddify上手更简单，非常适合新手使用。

Hiddify 可以在 Android、iOS、Windows、macOS 和 Linux 等系统平台上使用，支持 Shadowsocks/Vless/Vmess/Trojan/Reality/TUIC/Wireguard/Hysteria/SSH 等超多代理协议，能很好地保护用户数据隐私安全。据说，Hiddify由伊朗人开发（未经验证），支持英语、波斯语、汉语、俄语、阿拉伯语、西班牙语和葡萄牙语等8种主流语言，功能非常全面，也非常强大，例如自动选择节点、TUN 模式、使用远程配置文件等高级功能。

3.2 一键导入到Hiddify客户端

首先，我们登录xgCloud官方网站，找到复制订阅地址处，点击“一键订阅”，然后在弹出的窗口中，点击“导入到 Hiddify Next”按钮，自动唤起Hiddify客户端，并导入订阅信息，请稍等片刻。

导入成功后，如下图所示：

我们点击“点击连接”按钮，会自动下载订阅节点信息，并自动随机选择一个节点连接。我们可以在客户端“代理”界面，然后选择自己需要的节点使用。
如果我们使用一键订阅导入Hiddify客户端失败，或无法成功获取到订阅节点，那么我们可以使用通用订阅地址获取最新的节点信息。

3.2.1 复制通用订阅地址，并导入到Hiddify客户端

首先，我们登录xgCloud官方网站，找到复制订阅地址处，点击“一键订阅”，在弹出的窗口中，点击“复制订阅地址”，即可复制通用订阅地址到剪贴板。

3.2.2 打开Hiddify客户端，添加订阅地址

我们打开Hiddify客户端，点击主界面的“+ 新的配置文件”，在弹出窗口中，选择”从剪贴板添加“或”手动输入“。如果选择“从剪贴板添加”，效果跟直接一键导入相似。如果我们选择“手动输入”，那么请在弹出的配置文件窗口中填写相关信息，并点击“自动更新”，然后设置自动更新的时间间隔，最后点击“保存”。如下图所示：

此时，我们可以点击左上角的“更新配置文件”按钮，然后点击“点击连接“按钮即可，会自动下载节点信息，并随机连接一个节点，进行代理上网。如下图所示：

3.3 Hiddify客户端配置和通用设置

3.3.1 Hiddify客户端配置

一般情况下，我们保持默认即可。如果有特殊需求，可以在”配置选项“界面进行单独配置。我们切换到配置选项界面，建议”服务模式“选择系统代理。如果您所有连接节点均超时，全部显示Timeout，那么一般是本地网络DNS解析的问题。此时，您可以尝试将直连DNS修改为 114.114.114.114 或 223.5.5.5 或 119.29.29.29，一般就可以正常解析了。

3.3.2 Hiddify客户端设置

一般情况下不要做特殊设置，在Hiddify客户端设置界面可以选择语言，设置开机是否启动，以及是否静默启动等，按需求设置，其他保持默认即可。

至此，这篇Hiddify教程就完结了。如果还有其他问题，请再折腾几次，熟悉之后就好了。
